# Aplikasi Keuangan Rumah

Aplikasi sederhana untuk mencatat pemasukan dan pengeluaran keuangan rumah tangga.

## Fitur

- Mencatat pemasukan
- Mencatat pengeluaran
- Melihat saldo
- Riwayat transaksi

## Tujuan

Aplikasi ini dibuat untuk membantu mengelola keuangan rumah tangga agar lebih teratur.

## Teknologi

- HTML
- CSS
- JavaScript

## Cara Menjalankan

1. Download repository ini
2. Buka folder project
3. Jalankan file index.html di browser
